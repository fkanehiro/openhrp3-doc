ユーザインタフェース概要

OpenHRPを構成する各種要素について説明します。

GrxUIを起動すると下図のようなウインドウとなります（バージョンにより多少画面の構成等が異なります）。

    1. Item View
    2. Logger View
    3. Start Simulationボタン　…　シミュレーションを開始／停止します。
    4. GrxUI メニュー
    5. NameService Monitor View
    6. Process Manager View
    7. Server Manager View
    8. Graph View
    9. Property View
    10. Robot State View
    11. GrxUIのパースペクティブを開いていることを表します。
    12. 3D View
    13. Simulation View
    14. Controller View
    15. Collision Pair View
    16. Text Editor View
    17. JythonPrompt View　…　初期状態では表示されません。
    18. Range Sensor View　…　初期状態では表示されません。
    19. Path Planning View …　初期状態では表示されません。「移動動作設計チュートリアル」を参照してください．

ウインドウ(View)の操作

  • ウインドウ(View)の枠をドラッグしてサイズを変更することができます。
  • ウインドウ(View)をドラッグ＆ドロップして位置を変更することができます。
  • ウインドウ(View)をドラッグ＆ドロップして外に出すと独立したウインドウになります。
  • 表示されていないウィンドウを開くには、ウィンドウメニューから、ビューの表示　－　その他　を選択し、表示したいビューを選択します。
