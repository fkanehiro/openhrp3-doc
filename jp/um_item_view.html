Item View概要

Item Viewを構成する各種要素について説明します。

    1. プロジェクトホルダー
      シミュレーションの設定を管理している各アイテムが保存されています。
      右クリックすることで、新規作成、読み込み、保存等ができます。プロジェクト名を変えたいときは右クリック→"名前を付けて保存"によって別名で保存することにより行います。
      プロジェクトはxml形式でファイルに保存されます。
    2. Simulation ホルダー
      Simulationアイテム用のホルダーです。
      右クリックすることで作成、全アイテム削除ができます。複数のSimulationアイテムを持つことができますが、同時に有効なのは１つだけです。
    3. Simulation アイテム
      シミュレーションの時間や積分方法などを管理するアイテムです。
      右クリックすることでプロパティの復元、名前の変更、削除ができます。
      "シミュレーション View"で内容を編集することができます。プロパティの復元を実行すると、プロジェクトに保存された設定に戻すことができます。
    4. Model ホルダー
      Modelアイテム用のホルダーです。 右クリックすることで新規作成、読み込み、全アイテム削除ができます。複数のModelを持つことができます。
      新規作成したモデルアイテムは、"GrxUIを用いたモデル作成"を用いて編集することができます。
    5. Model アイテム（ロボット）
      モデルを記述したVRMLファイルです。ロボットの表示は、コントローラを持つことができるモデルを示します。
      右クリックすることで名前の変更や再読み込み、環境モデルへの変更等ができます。
    6. Model アイテム（環境）
      モデルを記述したVRMLファイルです。環境の表示は、コントローラを持つことができないモデルを示します。
      右クリックすることで名前の変更や再読み込み、ロボットモデルへの変更等ができます。
    7. World State ホルダー
      World Stateアイテム用のホルダーです。
      右クリックすることで作成、読み込み、全アイテム削除ができます。複数のWorld Stateアイテムを持つことができますが、同時に有効なのは１つだけです。
    8. World State アイテム
      シミュレーションのログを管理するアイテムです。
      右クリックすることでログの読み出しや保存ができます。
    9. Collision Pair ホルダー
      Collision Pairアイテム用のホルダーです。
      右クリックすることで全アイテム削除ができます。複数のCollision Pairを持つことができます。
      Collision Pairは"コリジョン View"で追加、編集ができます。
    10. Collision Pair アイテム
      衝突計算を行うモデルの組み合わせを管理するアイテムです。
      右クリックすることで名前の変更、削除などができます。
    11. Graph Contents ホルダー
      Graph Contentsアイテム用のホルダーです。
      右クリックすることで作成、全アイテム削除ができます。複数のGraph Contentsを持つことができますが、同時に有効なのは１つだけです。
    12. Graph Contents アイテム
      グラフ表示の内容を管理するアイテムです。
      右クリックすることで名前の変更、削除などができます。 Graph Contentsは"グラフ View"で編集できます。
    13. Python Script ホルダー
      Python Scriptアイテム用のホルダーです。
      右クリックすることで作成、読み込み、全アイテム削除ができます。複数のPython Scriptを持つことができますが、同時に有効なのは１つだけです。
    14. Python Script アイテム
      Python Scriptファイルです。
      右クリックすることで削除、保存などができます。
      有効なPython Scriptは"テキストエディタ View"で編集できます。
    15. PPAlgorithm ホルダー
      PPAlgorithm アイテム用のホルダーです。
      右クリックすることで作成、全アイテム削除ができます。複数のPPAlgorithmを持つことができますが、同時に有効なのは１つだけです。
    16. PPAlgorithm アイテム
      Path Planningパラメーターを管理するアイテムです。
      右クリックすることで名前の変更、削除などができます。
      PPAlgorithmは"プロパティ View"で編集できます。
    • アイテムをダブルクリックすることで、有効/無効を切り替えることができます。無効の場合アイテム名が"(アイテム名)"のように括弧付きになります。
    • アイテム選択すると"プロパティ View"に詳細が表示されます。
