対応OS

現在のところ、OpenHRP3は以下の環境で動作確認を行っています．

  • OS
    • Ubuntu Linux 10.04LTS ～12.10
    • Windows XP, 7
  • OpenRTM-aist 1.0.0
  • OpenRTM-aist 1.1.0
  • Eclipse 3.4SR2

Linuxに関しては、Ubuntu以外のディストリビューションの多くで動作させることが可能だと思われます. 動作実績のあるOSについては下記のページをご覧ください.

  • OS/ディストリビューション 対応状況

ただし、本サイトのマニュアルや配布パッケージの構成などは公式対応OSを前提としていますので、 それ以外のOS上で使用する際には、OSの操作やプログラミングに関してより高度な知識を求められる場合があります。

なお、使い勝手と動作速度の点でWindowsよりもLinux上での使用をおすすめ致します。
