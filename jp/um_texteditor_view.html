Text Editor View概要

Text Editorを構成する各種要素について説明します。


Text Editor View

    1. Save　ボタン
      ファイルを保存します。
    2. Save as ボタン
      名前を付けて、ファイルを保存します。
    3. Editor　エリア
      ファイルを編集できます。
    4. 行番号表示
      現在のカーソル位置を表示します。
