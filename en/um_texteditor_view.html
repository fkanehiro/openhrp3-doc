Text Editor View

Here we explain the various elements that compose Text Editor View.


Text Editor View

    1. Save button
      A file is saved.
    2. Save as button
      A file changes a name and is saved.
    3. Editor area
      A file is edited.
    4. The display of a row number
      The present cursor position is displayed.
