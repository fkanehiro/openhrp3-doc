Overview of User Interface

Here we describe the various elements that compose OpenHRP user interface (so called GrxUI).

When GraUI is started, you will get a window as shown in following figure.

    1. Item View
    2. Logger View
    3. Start Simulation button - (Used to start/pause simulation)
    4. GrxUI menu
    5. NameService Monitor View
    6. Process Manager View
    7. Server Manager View
    8. Graph View
    9. Property View
    10. Robot State View
    11. Express that GrxUI perspective is opened.
    12. 3D View
    13. Simulation View
    14. Controller View
    15. Collision Pair View
    16. Text Editor View
    17. JythonPrompt View - (Not viewed at initial state)
    18. Range Sensor View - (Not viewed at initial state)
    19. Path Planning View - (Not viewed at initial state) Please refer to "a path planner tutorial".

Operations in tab-view

  • You can change the size of the tab-views by dragging an edge.
  • Change the location of tab-views by drag-and-drop operation.
  • Drag-and-drop a tab-view out of the main window, so it will be popped-out as an individual window.
