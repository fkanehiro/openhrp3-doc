Graph View

Here we explain various elements that compose Graph View. Windows settings are taken for following example.

  1. Clicks to chooses a graph.
  2. If bar is dragged, the width of a graphical representation and height are changed.
  3. The range of the horizontal axis of all the graphs is set up.

    1. The range of time to display is set up.
    2. The position of a red line marker is set up.
    3. A setup is reflected in a graph.

  4. The range of the vertical axis of the chosen graph is set up.

    1. The maximum of a graph is set up.
    2. The minimum of a graph is set up.
    3. A setup is reflected in a graph.

  5. The contents displayed on the chosen graph are set up.

    1. A model is chosen.
    2. The kind of sensor is chosen.
    3. A sensor is chosen.
    4. A data item is chosen.
    5. The selected item is deleted.
    6. The set-up item is added.
    7. The color of a graph is chosen.
    8. The name of a graph is edited.
    9. A setup is reflected in Graf.
